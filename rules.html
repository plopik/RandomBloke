Mini-Infinite-Tennis - Game Rules

  • Each player have an energy pool that is used to make moves.
  • Each round each player choose how much energy to use
  • After each round each player recover 1 energy.
  • Each round one of the player is randomly awarded a point
  • The first player to reach 3 points wins the set.
  • After a set, scores are reset and a new set begins.
  • There are N bots competing against a Dummy.
  • The dummy always use 1 energy.
  • Set win ratios and confidence intervals are displayed in the chart.
  • You can edit the strategy for the bots using the provided text box.

How is the Winner Chosen Each Round?

In each round, both the Player1 and player2 choose how much energy to use. The winner is chosen randomly, but the chance of winning is proportional to the energy spent: 

Let E1 = energy used by player 1

Let E2 = energy used by player 2

A random number R is chosen between 0 and (A + B).

If R < E1, player 1 wins the point.

Otherwise, player 2 wins the point.

This means the probability that player 1 wins is E1 / (E1 + E2).
